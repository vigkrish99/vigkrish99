## What I Build

### AI Agent Systems
- **Custom MCP Server (35+ tools)** — Purpose-built Model Context Protocol server for a production textile ERP. 79-column MS SQL schema with embedded gotcha documentation for schema inconsistencies. One of the earliest production MCP implementations for a real business.
- **Multi-Agent Orchestration (5 LLMs)** — Orchestrates Claude Opus, GPT-5.4, Grok 4.1, DeepSeek V3.2, and Gemini via OpenRouter. 6-phase pipeline: Planning → Distribution → Parallel Execution → Peer Review → Synthesis → Voting. Swappable skill configs via YAML.
- **ReAct Agent + WhatsApp Bot** — Multi-step reasoning agent generating dynamic SQL against a legacy ERP. Up to 10 reasoning steps per query with self-correction. Deployed as a WhatsApp bot — 7,100+ SKUs searchable via natural language.
- **Dual-Tier LLM Intelligence Platform** — Claude Haiku (fast screening) → Gemini Pro (deep analysis). Monitors Reddit, Twitter, job boards. Per-model cost tracking with auto-pause on budget thresholds. Generated a 10,044-datapoint VoC intelligence library.

- **Textile Business Intelligence** — AI WhatsApp bot + 17-report dashboard over legacy MS SQL ERP. ReAct agent, custom MCP server, 4-tier auto-sync, GraphQL API, 279 tests.
